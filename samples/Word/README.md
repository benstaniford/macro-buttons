# Microsoft Word MacroButtons Profile

Quick access to common Microsoft Word keyboard shortcuts optimized for touch-screen macro panels.

## Usage

1. Right-click MacroButtons tray icon → Profiles → Import Profile
2. Browse to this file: `samples/Word/microsoft-word.json`
3. The profile will auto-activate when Word is the active window

## Shortcuts Included

### Main Panel (20 shortcuts + clock)

**Styles:**
- Heading 1 (Ctrl+Alt+1) - Apply Heading 1 style
- Heading 2 (Ctrl+Alt+2) - Apply Heading 2 style
- Heading 3 (Ctrl+Alt+3) - Apply Heading 3 style
- Normal Style (Ctrl+Shift+N) - Return to normal paragraph style
- Apply Style (Ctrl+Shift+S) - Open style picker dialog

**Font Formatting:**
- Font Dialog (Ctrl+D) - Open font formatting dialog
- Increase Font Size (Ctrl+]) - Make text larger
- Decrease Font Size (Ctrl+[) - Make text smaller
- Change Case (Shift+F3) - Toggle UPPER/lower/Title Case
- Copy Format (Ctrl+Shift+C) - Copy formatting from selection
- Paste Format (Ctrl+Shift+V) - Apply copied formatting

**Paragraph Formatting:**
- Bullet List (Ctrl+Shift+L) - Create bulleted list
- Center Align (Ctrl+E) - Center text alignment
- Left Align (Ctrl+L) - Left text alignment
- Right Align (Ctrl+R) - Right text alignment
- Justify (Ctrl+J) - Justify text alignment

**Review & Collaboration:**
- Insert Comment (Ctrl+Alt+M) - Add a comment at cursor
- Track Changes (Ctrl+Shift+E) - Toggle track changes mode

**Document Navigation:**
- Insert Hyperlink (Ctrl+K) - Create clickable link
- Go To (Ctrl+G) - Jump to page, section, bookmark, etc.

## Features

- **Educational Format:** Each button displays the keyboard shortcut on the second line for easy reference
- **Auto-Activation:** Profile automatically activates when Word window is in focus
- **Touch-Optimized:** Large buttons perfect for touch screens and secondary displays
- **Retro Terminal Theme:** Green-on-black aesthetic with prominent clock display

## Customization

Edit the JSON file to:
- Add more Word-specific shortcuts (see Microsoft's official documentation)
- Rearrange button layout to match your workflow
- Change colors/themes (foreground, background)
- Adjust refresh interval for the clock
- Modify auto-activation window pattern

## Tips

- **Heading Shortcuts:** Use Ctrl+Alt+1/2/3 to quickly apply heading styles for document structure
- **Change Case (Shift+F3):** Press repeatedly to cycle through uppercase, lowercase, and title case
- **Format Painter:** Copy format (Ctrl+Shift+C) from styled text, then paste format (Ctrl+Shift+V) to other text
- **Apply Style Dialog:** Ctrl+Shift+S opens a quick picker - type the style name and press Enter
- **Track Changes:** Essential for collaborative editing - toggle on/off with Ctrl+Shift+E

## Requirements

- MacroButtons application installed
- Microsoft Word (any recent version)
- Windows with .NET 8 Desktop Runtime
